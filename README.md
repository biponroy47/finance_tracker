Simple little Finance Tracker web app built in 3 hours, no database.

Uses a Google Spreadsheet + Script for backend.

90% "vibe coded".

## Local setup

Copy `config.example.js` to `config.js` and set `SCRIPT_URL` to your Google
Apps Script web app `/exec` URL. `config.js` is ignored so the private endpoint
does not get committed.
